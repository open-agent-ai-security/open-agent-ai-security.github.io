# open-agent-ai-security.github.io

Landing site for the **Open Agentic and AI Security Community**, served at
<https://open-agent-ai-security.github.io/>.

A single self-contained page: `index.html` (inline CSS) plus `assets/` (logos,
the Exabeam sponsor mark, and a self-hosted, ISC-licensed GoatCounter
`count.js` for privacy-friendly, cookieless analytics — no third-party
executable JavaScript runs on the page).

## Editing

Edit `index.html` directly and push to `main`; GitHub Pages publishes from the
repository root. `.nojekyll` keeps Pages from running the files through Jekyll.

Project sites (e.g. Praxen at `/praxen/`) live in their own repositories and
are linked from the Projects section here.
